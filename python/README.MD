# A simple generator for Fibonacci Numbers
def fib(limit):
    # Initialize first two Fibonacci Numbers
    a, b = 0, 1
    # One by one yield next Fibonacci Number
    while a < limit:
        yield a
        a, b = b, a + b
# Create a generator object
x = fib(5)

print(next(x))
print(next(x))

Built-In Class Attributes : 
Every Python class keeps following built-in attributes and they can be accessed using dot operator like any other attribute:

 + `__dict__` âˆ’ Dictionary containing the class's namespace.

 + `__doc__ `âˆ’ Class documentation string or none, if undefined.

 + `__name__` âˆ’ Class name.

 + ` __module__` âˆ’ Module name in which the class is defined. This attribute is "`__main__`" in interactive mode.

 + `__bases__ `âˆ’ A possibly empty tuple containing the base classes, in the order of their occurrence in the base class list.
